For Local Setup:
1. Install Node.js ( Use Version 20 or Above )
2. cd to the project directory
3. Copy the .env file into the project directory
3. Run npm install
4. Run npm run start:dev
5. Open http://localhost:8080/api-docs in your browser for swagger documentation

For Production Setup:
1. Project has been deployed on AWS EC2 instance
2. Use the following link to access the application: http://65.2.63.196:8080/api-docs
